# Misuo Metallic Furniture - Interactive Sales Dashboard

## Project Overview
This project presents a comprehensive, interactive Sales Dashboard built entirely in **Microsoft Excel** using advanced data processing techniques and **VBA Custom Automation**. The dashboard analyzes real-world business data from **"Misuo Metallic Furniture"** (مسيو للأثاث المعدني) to extract actionable business insights, track financial metrics, and streamline operational workflows.

## Key Measurable Achievements & Metrics Analyzed
* **Total Sales Revenue Volume:** Generated over **EGP 1,179,845** in analyzed transactions.
* **Order Processing:** Tracked and structured **71 distinct bulk orders**.
* **Customer Acquisition:** Analyzed a database of **115 active clients** to monitor purchasing patterns.
* **Pricing Optimization:** Calculated an Average Selling Price (ASP) of **EGP 10,260** per unit.
* **Operational Efficiency:** Evaluated fulfillment timelines, analyzing delivery delay rates (47% delayed vs. 53% on-time) and cancellation impacts (17% failed/cancelled orders vs. 83% successful deliveries).

## Technical Implementation Workflow

### 1. Data Cleansing & Transformation
* Resolved severe data duplication, syntax errors, and missing transaction variables inside the raw dataset.
* Standardized structural formats and aligned fragmented columns into a clean, unified analytical data pipeline using Advanced Excel workflows.

### 2. Data Modeling & Core Metrics (KPIs)
* Formulated key business metrics optimized for furniture manufacturing and retail distribution.
* Segmented regional sales behavior to pinpoint top-performing geographic areas and key sales platforms.
* Maintained inventory and design breakdowns through specialized performance tracking (Best-selling products, and top-selling item colors like Black and Grey).

### 3. Analytical Architecture
* Engineered multiple dynamic **Pivot Tables** to summarize high-volume business operations into lightweight, digestible summary tables.
* Conducted cross-tabulations between product lines, distribution platforms, and order fulfillment states.

### 4. Interactive Dashboard & VBA Automation
* Built a sleek, custom user interface featuring coordinated visualization charts (Donut, Pie, and Bar charts).
* Integrated interactive **Slicers** for effortless filtering by "Address/Area" and "Sales Platform".
* Implemented backend **VBA (Visual Basic for Applications)** scripting logic to guarantee fully dynamic report filtering and seamless performance responsiveness.

## Technologies Used
* **Microsoft Excel** (Data Cleansing, Pivot Tables, Advanced Charting)
* **VBA** (Custom Automation & Interaction Logic)
